MySQL数据库崩溃
anlondon Lv5
 2021-03-07 21:55:00  
  • MySql 
 

错误：

某台服务器下的网站能访问，但是所有的网站数据库都不能连接

使用PHPmyadmin 提示
#2003 无法登录 MySQL 服务器

ERROR 2003(HYO00):Can’t connect to MySQL server on ‘localhost’(10061)

原因：

数据库崩溃

解决：

  • 重启数据库
  1. win：cmd中；(在没有数据库管理软件，且不愿意重启服务器时)

    若提示服务名无效，在MySql/bin/目录，执行命令：mysqld -install 即可
    卸载同上 mysqld --remove mysql

  2. Linux：
  3. 或者重启服务器(开关机的那种，前提是，各项服务设置过开机启动)
