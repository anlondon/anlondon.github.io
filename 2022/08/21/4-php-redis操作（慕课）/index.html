4-1 php链接redis

  • Redis类的命名空间是根
    因此在其他命名空间使用redis类时需要加上\根目录符号
  • phpredis的命令和参数基本和redis.io中的实际命令对应

  • 见GitHub - phpredis/phpredis: A PHP extension for Redis
  • Redis 列表(List) | 菜鸟教程

list因为谁先进谁先出的特点，适合做下单
用户下单后提示用户下单成功，而后台则插入待做事项list，等实际操作后台时获取待做事项并处理。
