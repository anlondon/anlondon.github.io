windows2008 R2忘记密码
anlondon Lv5
  2021-03-19 14:25:58 2021-03-19 14:25   2021-03-19 15:01:16  

问题

服务器：Windows2008 R2
远程时一直提示密码错误，于是手动修改远程密码

解决

阿里等第三方虚拟主机
  1. 登录云服务器管理控制台。
  2. 单击左侧导航栏中的实例，在实例列表找到您需要远程登录的目标实例，单击远程连接，在弹出的页面中选择发送远程命令（云助手）的方式。
  3. 输入重置密码命令，然后单击执行按钮
    net user administrator 123456@p
    上述命令为重置adminitrator用户密码为123456@p
  4. 之后即可使用重置的账号和密码进行远程连接
本地
  1. win+X唤出命令行窗口
  2. net user administrator 123456@p 重置密码
