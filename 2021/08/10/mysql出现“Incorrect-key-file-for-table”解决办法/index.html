出现问题

SQLSTATE[HY000]: General error: 126 Incorrect key file for table ‘.\voice\admin_user.MYI’; try to repair it

引擎是：MyISAM
数据库：mysql5.7.26

运行下面的命令修复即可
repair table tables_name;
