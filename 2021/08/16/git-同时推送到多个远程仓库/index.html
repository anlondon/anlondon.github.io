git 同时推送到多个远程仓库
anlondon Lv5
 2021-08-16 14:20:00  
  • Git 
 
  • Git 
 

git默认远程仓库名为origin

git remote add oginin xxxx.git
git remote set-url --add oginin yyyy.git

之后push时，就会自动推送到设置的两个远程仓库上
